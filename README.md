# esm-seed

A seed for simple express-mongo JSON API applications with user authentication

[![Build Status](https://travis-ci.org/divanvisagie/esm-seed.svg?branch=master)](https://travis-ci.org/divanvisagie/esm-seed)

## Development

### Setup
```bash
npm install
```

### Testing

```bash
npm test
```

### Running
```bash
docker-compose up
npm run
```
